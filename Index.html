Why Audio Sound Manager needs "Allow access to all files" permission on Android 11

Audio Sound Manager is Secret Gallery vault where you can hide your secret files (photo,video,audio and documents).

Explaination :

Because of the limitation of GooglePlay, on Android 11 and above devices, apps must require "Allow access to all files" permission to access the files in the external storage. The permission was granted through a permission request dialog on the Android 10 and lower devices.

App's core purpose is to encrypt files and folders Disk/Folder Encryption and Locking.


Here is the reason why Audio Sound Manager needs permission:

  1. Audio Sound Manager stores the encrypted files in .AudioManagerPlusJJ on external storage. By doing this, the encrypted files will stay there if Audio Sound Manager is uninstalled accidentally and after you install Audio Sound Manager again, Audio Sound Manager can restore the data for you.

  2. If Audio Sound Manager store user's hidden data on app's directory without All Files Access then user lost his data when user uninstalled or accidentally uninstalled app.

  3. Audio Sound Manager supports encrypting any type of files on your external storage. Without permission, it is impossible to add the files on external storage to Audio Sound Manager.

  4. The backup and restore function also needs permission.


With this permission, The purpose of Audio Sound Manager is to protect your private data.

And this is the official explanation: https://developer.android.com/training/data-storage/manage-all-files#all-files-access-google-play
